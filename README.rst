====================
Mk20 Labs Salt Model
====================

OpenStack Reclass models for Mk-based cloud for training and development.


Available stacks
================


Mk.20 basic testing lab
-----------------------

* 1 config node
* 3 control nodes
* 1 compute node


Mk.20 advanced testing lab
--------------------------

* 1 config node
* 3 control nodes
* 1 compute node
* 1 monitor node


Mk.20 expert testing lab
------------------------

* 1 config node
* 3 control nodes
* 2 compute nodes
* 1 monitor node
* 1 meter node
* 1 log node


Mk.20 basic StackLight lab
--------------------------

* 1 config node
* 3 control nodes
* 1 compute node
* 1 monitor node


Mk.20 advanced StackLight lab
-----------------------------

* 1 config node
* 3 control nodes
* 1 compute node
* 3 monitor nodes
